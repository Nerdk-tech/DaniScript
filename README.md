### ⚡️ The Ultra-Lightweight Utility Engine
**DaniScript** is a modern styling language built for speed. It transforms design tokens into atomic CSS classes, giving you total control without the bloat.

## 📖 Quick Syntax
```html
<!-- Style faster with the ds- prefix -->
<div class="ds-flex ds-bg-dark ds-p-6 ds-rounded-xl">
  <h1 class="ds-text-primary ds-font-bold">
    Hello DaniScript
  </h1>
</div>

```
